# Snakemake: A Minibook for New Bioinformaticians

A concept-first orientation to reproducible workflows with
[Snakemake](https://snakemake.readthedocs.io/), written for incoming graduate
students in our bioinformatics group. It assumes a couple of semesters of
undergraduate molecular/cell biology and chemistry and about a year of
undergraduate computer science — and builds everything else from there, using a
small RNA-seq pipeline as a running example.

See [`OUTLINE.md`](OUTLINE.md) for the full structure and design notes.

## How to read it

Chapters 1–6 are the core and should be read in order; 7–11 are the practices that
make a pipeline trustworthy; 12 ties it all together; the appendices are reference
material.
